How to ?

Run

  • To compile : make all
  • Before running : export LD_LIBRARY_PATH=.
  • To Run : ./http-server
  • Open the doc make doc

Get

  • Source files are in /src
  • Binary files are in /bin
  • Documentation files ar in /doc_html
